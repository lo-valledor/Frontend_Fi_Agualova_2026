# Contribuir a ENERLOVA

Este documento explica cómo colaborar en el frontend de ENERLOVA para que las contribuciones sean consistentes, legibles y fáciles de mantener por otros desarrolladores.

1. Flujo de trabajo
   - Crea una rama a partir de `main` con nombre corto y descriptivo: `feat/usuario-login`, `fix/tabla-paginacion`.
   - Abre Pull Requests hacia `main` con una descripción clara del cambio, capturas de pantalla si aplica y pasos para reproducir.
   - Incluye pruebas unitarias o de integración cuando sea relevante.

2. Estilo y calidad
   - Mantén TypeScript estricto: usa tipos explícitos cuando sea necesario.
   - Sigue las reglas de ESLint y Prettier definidas en el repo. Ejecuta `pnpm lint` antes de push.
   - Prefiere hooks y componentes funcionales. Evita lógica en JSX; extrae helpers cuando la función crece.

3. Código generado / contenido asistido
   - Está permitido usar asistencia (snippets, ChatGPT, Copilot) para acelerar tareas, pero TODO código debe ser revisado y adaptado para que refleje el estilo del equipo.
   - No subir comentarios o marca visibles que indiquen que el texto/código fue generado por IA. Ejemplos a evitar en commits o archivos: "Generated by", "Copilot", "Assistant", "GPT", "LLM".
   - Añade un comentario en el PR si usaste AI para prototipar (ej.: "Se usó asistencia para esqueleto de funciones"), y explica los cambios manuales realizados.

4. Documentación
   - Documenta componentes y servicios usando los templates en `docs/templates/`.
   - Para componentes, añade ejemplos de uso y prop-types/Typescript types.

5. Revisión y aprobaciones
   - Pide al menos una revisión de un compañero.
   - Los reviewers deben validar: funcionalidades, seguridad (tokens/URLs), accesibilidad mínima y que no haya texto que delate generación automática.

6. Scripts útiles
   - `pnpm install` — instalar dependencias
   - `pnpm dev` — correr servidor de desarrollo
   - `pnpm build` — construir para producción
   - `pnpm lint` — linter
   - `pnpm test` — tests (si aplica)

Gracias por contribuir.
